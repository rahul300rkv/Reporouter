Repo Router Kya Hai?

Agar aap ek developer ho aur roz GitHub repos explore karte ho, toh aapne already gitmcp.io, gitingest.com, deepwiki.com aur uithub.com ke baare mein suna hoga. Manually URL mein github.com ko replace karna thoda time-waste hai — isiliye Repo Router bana hai. Bas apna repo link paste karo, destination select karo, aur ek click mein seedha sahi jagah pohonch jao. Koi installation nahi, koi signup nahi — sab kuch aapke browser mein hi process hota hai.

MCP Server Claude, Cursor, ya kisi bhi AI coding assistant ke liye perfect hai. Dir Tree poora codebase LLM-friendly format mein deta hai. DeepWiki AI-generated documentation banata hai. UIthub ek alternate LLM-context viewer hai. Aur ab aap apne README mein ek badge bhi laga sakte ho jo directly yahan deep-link karta hai!

FAQ — Aksar Puche Jaane Wale Sawaal

GitMCP kya hota hai?

GitMCP ek free service hai jo kisi bhi public GitHub repo ko ek Model Context Protocol (MCP) server mein badal deta hai, taaki Claude, Cursor jaise AI tools us repo ko directly samajh sakein aur reference de sakein.

Gitingest se kya milta hai?

Gitingest aapke repo ka clean directory structure aur file content ek readable, LLM-friendly text format mein generate karta hai, jisse poora codebase samajhna easy ho jaata hai.

Repo Router free hai kya, koi signup chahiye?

Haan, Repo Router bilkul free hai aur koi signup ya login nahi chahiye. Sirf GitHub link paste karo aur ek click mein redirect ho jao.

Kya mera code kahin upload ya store hota hai?

Nahi. Repo Router sirf URL ko transform karta hai aur ye sab kuch aapke browser mein locally hota hai. Koi data server pe bheja nahi jaata.

Kya private repos ke saath kaam karega?

Repo Router sirf link redirect karta hai — private repos ke liye aapko destination site pe authentication karni hogi, jaisa unki site par instruct kiya gaya ho.

MCP Integration aur Directory Structure mein kya difference hai?

MCP Integration (gitmcp.io) AI assistants ke liye hai jo aapke repo ko tool ke through query karna chahte hain. Directory Structure (gitingest.com) un logo ke liye hai jo jaldi se repo ka layout aur code samajhna chahte hain.

README badge kya hota hai aur kaise use karein?

Jab aap koi valid GitHub repo paste karte ho, Repo Router ek markdown badge snippet generate karta hai. Use apne repo ke README mein paste karo — badge click karne pe seedha Repo Router pe aa jaate hain, repo pre-filled ke saath. Ye aapke repo visitors ke liye ek shortcut ban jaata hai.

Sab kuch aapke browser mein local process hota hai, kahin upload nahi hota.
